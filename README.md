# css-length-units
List of CSS [`<length>`](https://developer.mozilla.org/en-US/docs/Web/CSS/length) units.

## Installation

```
npm install css-length-units [--save[-dev]]
```

## Usage

```js
var cssLengthUnits = require('css-length-units');
cssLengthUnits; /*
[
	"em",
	"ex",
	"ch",
	"rem",
	"vh",
	"vw",
	"vmin",
	"vmax",
	"px",
	"mm",
	"cm",
	"in",
	"pt",
	"pc",
	"mozmm"
]
*/
```
